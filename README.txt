av Gruppsex

Fucka inte ur repon, ladda bara upp sånt som är färdigt, blir lättast så.

För en .gitignore fil där ni skriver:
.gitignore
*~

git man, se manual för git typ
git help 'command', se hjälp för nåt command

git add -A, adda alla filer till kön
git commit -m "message", commita era add:ade ändringar till era lokala repo
git commit -am "message", gör båda ovan

git push origin, lägg upp ert commitade arbete på servern i mastergrenen, lägg till namn på gren om nu inte vill pusha till master
git pull origin, hämta senaste versionen från servern

git branch 'name', skriv namnet på er gren istället för name
git checkout 'branch', gå till er branch, skriv namnet på er gren istället för branch

För att merge:a en gren till en annan, gå till den gren ni kommer merge:a med, ex. merge:a 'Alex' med master
git checkout master, gå till master
git merge Alex, kanske behöver fixa till lite konflikter

git branch -d 'branch', ta bort en gren
